# Share Music

## サイト概要
### サイトテーマ
好きな曲やアーティストの感想を投稿したり、閲覧したりできるコミュニティサイト

### テーマを選んだ理由
自分の好きな音楽関係の何かを作ろうと思ったのがきっかけでした。自分が聞いた曲の感想を話したりするのが好きなのですが、音楽に特化した感想をシェアできるサービスがあったら便利だと思いこのテーマにしました。

### ターゲットユーザ
音楽鑑賞が好きな人

### 主な利用シーン
自分の好きな曲の感想を投稿するとき、他の人の感想を閲覧するとき
